Présentation

L'ordinateur PV16SOG possède un interpréteur BASIC inspiré de QBASIC. Le code source BASIC est d'abord compilé en bytecode pour être exécuter sur une machine virtuelle à piles. Ce document décris ce langage BASIC et son utilisation.

Caractéristques du langage.

  • Types de données
    • Entier 16 bits
    • Entier 8 bits (octets)
    • Chaîne de caractères ASCII
    • Tableau unidimensionnel de ces 3 types de données
  • Structures
    Il s'agit d'un langage procédural basé sur les fonctions déclarées avec le mot réservé FUNC ainsi que des sous-routines déclarées avec le mot résservé SUB. Il n'y a pas de numéros de lignes ni de GOTO ou GOSUB comme dans les versions plus ancienne de BASIC.
